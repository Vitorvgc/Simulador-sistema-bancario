## Uso

Para usar esse projeto:

* Clone o repositório;
* Instale o [JavaFX](https://docs.oracle.com/javafx/2/installation/jfxpub-installation.htm);
* Configure o JavaFX em sua IDE;
* Abra o projeto com a sua IDE;
* Execute o projeto;
